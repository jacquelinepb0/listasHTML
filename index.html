Passos para clonar um repositório

  1. Cria um repositório
  2. Copia a URL do repositório
  3. Abre o terminal
  4. Dá o comando: git clone "http://..."
  5. Muda de pasta: cd nome_do_repositorio
  6. Da o comando para entrar no vscode: code .

Após atualizar o codigo

  1. Git add
  2. Git commit -m "descricao_do_conteudo
  3. Git push origin main
